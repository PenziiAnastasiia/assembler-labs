longop.asm -- файл з процедурами, що оперують великими числами, розбиваючи їх на групи по 4 байт. Там представлені операції додавання, віднімання, множення і ділення, копіювання даних; а також побітове І між масивом і маскою на ділянці довжиною M, починаючи з біта N.

module.asm -- файл з процедурами, що представляють двійкові числа шістнадцятковими текстовими символами.

main4.asm - main8.asm -- файли лабораторних робіт, в яких використовувались ці процедури для обчислень та демонстрації результатів.
